## VSCode

### 中文匹配正则

要匹配中文字符

```regex
[\u4e00-\u9fa5]
```

匹配连续的中文字符

```regex
[\u4e00-\u9fa5]+
```

### Setting sync id

请复制这个 ID 并将其用于其他设备来下载配置。
